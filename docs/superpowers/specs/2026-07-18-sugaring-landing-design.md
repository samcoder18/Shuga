# Лендинг студии шугаринга — спецификация

Дата: 2026-07-18
Статус: утверждено пользователем

## Цель

Одностраничный лендинг для студии шугаринга в светлых карамельно-бежевых тонах. Язык контента — русский. Сайт должен открываться локально двойным кликом по `index.html`, без сборки и внешних зависимостей (кроме Google Fonts с фолбэком).

## Стек и структура файлов

Без фреймворков и бандлеров — HTML5 + CSS3 + ванильный JS (ES6+).

```
index.html      — семантическая разметка всех секций
styles.css      — стили: CSS-переменные, flex/grid, mobile-first адаптив
script.js       — весь интерактив, без библиотек
```

## Секции страницы (порядок на странице)

1. **Шапка (header)** — логотип «Сахар», якорная навигация (Услуги, Вопросы, Контакты), кнопка «Записаться». Липкая (`position: sticky`), тень появляется при скролле.
2. **Герой (hero)** — заголовок-оффер, подзаголовок, две CTA-кнопки: WhatsApp и Telegram. Декор — карамельные градиентные «капли»/формы на CSS, без растровых изображений.
3. **Услуги и цены** — сетка карточек по зонам: ноги, руки, подмышки, бикини, комплексы. В каждой карточке: название, краткое описание, цена-заглушка (диапазон ~800–2500 ₽), кнопка «Записаться» (ссылка на мессенджер).
4. **FAQ** — аккордеон ровно из 5 вопросов: «Это больно?», «Как подготовиться к процедуре?», «Сколько держится эффект?», «Какой длины должны быть волосы?», «Какие есть противопоказания?».
5. **Контакты** — адрес-заглушка, часы работы, телефон-заглушка `+7 (900) 000-00-00`, кнопки WhatsApp / Telegram, стилизованная карта-заглушка на CSS (без внешних картографических сервисов).
6. **Футер** — копирайт, ссылки на соцсети-заглушки.

## Дизайн-система

- **Палитра (CSS-переменные):**
  - фон: `#FDF9F4` (сливочный)
  - акцент карамельный: `#C8956C`, тёмный вариант `#A9714B`
  - текст: `#3E2F25` (тёмно-коричневый)
  - карточки: белые `#FFFFFF`
- **Шрифты:** Playfair Display (заголовки) + Manrope (текст) через Google Fonts (`font-display: swap`); фолбэк — системные serif/sans-serif, чтобы страница работала офлайн.
- **Стиль:** скруглённые карточки (border-radius ~16–24px), мягкие тени, плавные hover-переходы, декоративные градиентные формы.
- **Адаптив:** mobile-first; контрольные ширины 375px (моб.), 768px (планшет), 1280px (десктоп).

## Интерактив (script.js)

- FAQ-аккордеон (один открытый вопрос за раз или независимые — выбрать независимые, простое поведение)
- Бургер-меню на мобильной ширине
- Плавный скролл по якорям
- Fade-in анимация появления секций через IntersectionObserver
- Тень у липкой шапки при скролле

## Контент

Все тексты — реалистичные русские заглушки, которые владелец заменит на свои:

- Название студии: «Сахар»
- Телефон: `+7 (900) 000-00-00`, ссылки `https://wa.me/79000000000`, `https://t.me/username` (заглушки)
- Адрес: «г. Москва, ул. Примерная, д. 1»
- Цены в рублях, правдоподобные для рынка

Тексты сосредоточены в `index.html` и оформлены так, чтобы их было легко найти и заменить.

## Фото и медиа

Без внешних изображений: вся графика — CSS-градиенты, формы, эмодзи/инлайн-SVG-иконки. Сайт полностью работает офлайн.

## Проверка результата

- Открыть страницу в браузере (через локальный HTTP-сервер или `file://`)
- Скриншоты на ширинах 375px и 1280px, визуальная проверка
- Проверить: аккордеон FAQ, бургер-меню, плавный скролл, анимации появления, отсутствие ошибок в консоли

## Что НЕ входит (YAGNI)

- Бэкенд, формы заявок, онлайн-запись
- Внешние фото, виджеты карт, слайдеры отзывов
- Мультиязычность, тёмная тема
- Сборка, фреймворки, npm-зависимости
